Helpful Tips

Always call if needed!!!!!

555-555-5555


  • Lights: Always turn off the breaker, but if not feasible, consider these safety steps:
    1. use a wooden ladder

    2. wear no jewelery

    3. touch no metal

    4. have an assistant if available

    5. no one touching you if they're on ground
  • Plugs: Always turn off the breaker, but if not feasible, consider these safety steps:
    1. use a wooden platform to kneel on

    2. wear no jewelery

    3. touch no metal

    4. remain grounded, do not sweat on plug

    5. no one touching you if they're on ground
  • Fans: Always turn off the breaker, but if not feasible, consider these safety steps:
    1. use a wooden ladder

    2. wear no jewelery

    3. touch no metal

    4. have an assistant if available

    5. no one touching you if they're on ground
